Comments API

Creating a comment for post

POST /api/comments

Parameters

Name Description
comment[author] Author of comment
comment[content] Content of comment
category_id ID related category
post_id ID comentable post

Request

Headers

Content-Type: application/json
Host: example.org
Cookie:

Route

POST /api/comments

Body

{"comment":{"author":"John Taylor.","content":"better comment"},"category_id":null,"post_id":1}

cURL

curl "http://localhost:3000/api/comments" -d '{"comment":{"author":"John Taylor.","content":"better comment"},"category_id":null,"post_id":1}' -X POST \
	-H "Content-Type: application/json" \
	-H "Host: example.org" \
	-H "Cookie: "
